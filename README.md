# 🎯 Career Command Center V8

공기업·금융권 188개 기관 9등급 전수 평가 대시보드

## Features
- **9등급 채점 체계** — DNA 적합도(50) + 진입 가능성(30) + 현실 보정(20) = 100점
- **188개 기관 상세 평가** — 클릭 시 기관별 코멘트 펼침
- **다중 필터** — 등급·분류·지역 중복선택 가능
- **콤팩트 뷰** — 기관명+등급만 빠르게 조회
- **18개월 로드맵** — 2026.02~2027.06 월별 실행 계획
- **DNA 5코드 분석** — 🏭현장 📊분석 🔨빌더 📚교육 🌍탐색
